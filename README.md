# Work Day Scheduler

## Purpose
A website that offers a color coded work day scheduler.  Time blocks change color depending on past(grey), present(red), and future(green).  Tasks save in local storage and remain on page after refresh.

## Built With
* HTML
* CSS
* Javascript
* JQuery
* Bootstrap
